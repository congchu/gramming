내가 지원하려는 회사 파악하기

  1. 어떤 회사들이 있을까?(1) 규모별 - 중소/중견/대기업
  2. 어떤 회사들이 있을까?(2) 고객별 = SI/SM/서비스/솔류션
  3. 회사는 어떤 일들을 할까? - 회사가 하는 일 엿보기 구경하는 방법 :)

자소서 팁

  2. 자소서 쓰기 팁(2) - 용어를 정확히 써야해요
  3. 자소서 쓰기 팁(3) - 장점을 팍팍! 강조해주세요

포트폴리오 팁

  1. 포트폴리오 만들기 팀(1) - 포폴에 정답은 없다
  2. 포트폴리오 만들기 팁(2) - 읽는 사람을 배려한 포트폴리오 만들기
  3. 포트폴리오 만들기 팁(3) - Github 활용법

면접 팁

  1. 면접 팁(1) - 자신감을 장착하자
  2. 면접 팁(2) - 단골 질문 답변 완성해두자
  3. 면접 팁(3) - 강점 기반의 캐릭터를 만들어 두자!
  4. 면접 팁(4) - 스토리 Pool을 만들자
  5. 면접 팁(5) - 실전에서 배우자
